# 🔑 우리학교 키박스

> 학교 업무에 자주 쓰는 사이트 계정, 학교 정보, 은행 계좌, 결제 수단을
> **한 곳에서 빠르게 찾고 복사할 수 있는 학교 행정실용 로컬 키박스**입니다.

[👉 바로 사용하기](https://sen-vip.github.io/school-keybox/)

---

## 📌 소개

**우리학교 키박스**는 학교 행정실에서 자주 사용하는 정보를
흩어지지 않게 정리하고, 필요한 순간 빠르게 검색·복사할 수 있도록 만든 웹 도구입니다.

사이트 계정, 비밀번호, 학교 기본정보, 은행 계좌, 카드 정보 등을
업무 흐름에 맞게 정리할 수 있으며, 모든 데이터는 서버로 전송되지 않고
**사용 중인 PC 브라우저에만 저장**됩니다.

---

## ✨ 주요 기능

### ⚡ 빠른복사

자주 쓰는 항목을 빠른복사 영역에서 바로 확인할 수 있습니다.

* 자주 쓰는 ID·PW 빠른 복사
* 버튼 클릭 시 자동 복사
* 즐겨찾기 항목 중심으로 빠른 접근
* 반복 로그인 업무 시간 단축

---

### 🌐 사이트 계정 관리

학교 업무용 사이트 계정을 한 곳에 정리할 수 있습니다.

* 사이트명
* 아이디
* 비밀번호
* URL
* 메모
* 즐겨찾기 여부

ID와 비밀번호는 각각 버튼으로 복사할 수 있어
로그인할 때마다 문서를 찾거나 직접 입력할 필요를 줄여줍니다.

---

### 🏫 학교 정보 관리

자주 입력하는 학교 기본 정보를 정리할 수 있습니다.

예시 항목:

* 학교명
* 기관번호
* 사업자등록번호
* 대표 전화번호
* 주소
* 담당자 정보
* 기타 학교 공통 정보

각종 신청서, 계약서, 공문, 시스템 입력 시 반복 입력하는 정보를
빠르게 확인하고 복사하는 데 사용할 수 있습니다.

---

### 🏦 은행 계좌 관리

학교 업무에 사용하는 계좌 정보를 정리할 수 있습니다.

* 은행명
* 계좌번호
* 예금주
* 사용 목적
* 메모

지출, 수입, 계약, 정산 업무에서 자주 확인하는 계좌 정보를
한눈에 정리할 수 있습니다.

---

### 💳 결제 수단 관리

학교 업무용 카드나 결제수단 정보를 정리할 수 있습니다.

* 카드명
* 카드번호
* 사용처
* 담당자
* 메모

업무용 결제수단을 구분해 관리할 수 있어
필요한 정보를 빠르게 확인할 수 있습니다.

---

## 🔍 검색 기능

상단 검색창에서 필요한 정보를 바로 찾을 수 있습니다.

검색 대상 예시:

* 사이트명
* 아이디
* 메모
* 학교 정보 항목
* 계좌 정보
* 결제 수단 정보

정보가 많아져도 검색으로 빠르게 찾을 수 있어
초보자도 쉽게 사용할 수 있습니다.

---

## 🔐 저장 방식

우리학교 키박스는 **서버 저장 방식이 아닙니다.**

입력한 데이터는 외부 서버로 전송되지 않으며,
사용 중인 PC의 브라우저 저장공간에만 저장됩니다.

### 저장 관련 기능

* 이 PC에 저장
* PC 저장내용 불러오기
* 저장정보 초기화
* JSON 백업
* JSON 불러오기

중요한 정보는 PC 변경, 브라우저 초기화, 캐시 삭제 등에 대비하여
정기적으로 JSON 백업을 해두는 것을 권장합니다.

---

## 👀 비밀번호 표시 옵션

비밀번호는 필요에 따라 표시 방식을 바꿀 수 있습니다.

* 그대로 표시
* 마스킹 표시

공용 공간이나 다른 사람이 볼 수 있는 환경에서는
마스킹 표시 상태로 사용하는 것을 권장합니다.

---

## 📊 엑셀 기능

엑셀을 활용해 데이터를 관리할 수 있습니다.

### 지원 기능

* 기본양식 다운로드
* 엑셀 업로드
* 엑셀 다운

기본양식에 맞춰 정보를 작성한 뒤 업로드하면
웹 화면에서 정리된 형태로 확인할 수 있습니다.

엑셀 기능은 여러 항목을 한 번에 정리하거나
기존 자료를 옮겨올 때 유용합니다.

---

## 🖨️ 인쇄 / PDF 저장

정리된 내용을 인쇄하거나 PDF로 저장할 수 있습니다.

* 인쇄용 출력
* PDF 저장
* 종이 보관용 자료 작성
* 내부 업무 참고자료 출력

단, 보안이 필요한 정보가 포함될 수 있으므로
출력물 관리에 주의해 주세요.

---

## 🧭 사용 방법

### 1. 사이트 접속

아래 주소로 접속합니다.

https://sen-vip.github.io/school-keybox/

---

### 2. 필요한 항목 입력

사이트 계정, 학교 정보, 은행 계좌, 결제 수단을 입력합니다.

---

### 3. 이 PC에 저장

입력한 내용을 보관하려면
상단의 **이 PC에 저장** 버튼을 누릅니다.

---

### 4. 필요할 때 검색·복사

검색창에서 필요한 항목을 찾고,
ID·PW 또는 필요한 정보를 복사해 사용합니다.

---

### 5. 백업하기

중요한 정보는 **JSON 백업**으로 따로 보관하는 것을 권장합니다.

---

## 💡 이런 경우에 유용합니다

* 학교 업무용 사이트 계정이 여러 개일 때
* 공통 ID·PW를 자주 찾아야 할 때
* 학교 기본정보를 반복 입력해야 할 때
* 은행 계좌나 결제수단 정보를 자주 확인해야 할 때
* 엑셀이나 메모장에 흩어진 정보를 한 곳에 모으고 싶을 때
* 신규 담당자도 쉽게 확인할 수 있는 업무용 키박스가 필요할 때

---

## ⚠️ 사용 전 확인사항

이 도구는 학교 행정실 업무 편의를 위한 보조 도구입니다.

다음 사항을 꼭 확인해 주세요.

* 공용 PC에서는 사용 후 저장정보를 반드시 확인해 주세요.
* 중요한 정보는 JSON 백업 파일로 따로 보관해 주세요.
* 브라우저 캐시 삭제, PC 교체, 브라우저 변경 시 저장정보가 사라질 수 있습니다.
* 비밀번호가 포함된 출력물이나 백업 파일은 안전하게 관리해 주세요.
* 기관 보안 지침이 있는 경우 해당 지침을 우선해 주세요.

---

## 🛡️ 보안 안내

우리학교 키박스는 별도의 서버나 데이터베이스를 사용하지 않습니다.

* 회원가입 없음
* 로그인 없음
* 서버 전송 없음
* 외부 저장 없음
* 현재 PC 브라우저 중심 저장

다만, 입력한 정보 자체가 민감할 수 있으므로
사용자 PC 보안과 백업 파일 관리는 사용자가 직접 주의해야 합니다.

---

## 🧑‍💻 제작 의도

학교 행정실에서는 사이트 계정, 기관 정보, 계좌, 결제수단처럼
자주 쓰지만 여기저기 흩어지기 쉬운 정보가 많습니다.

이런 정보를 매번 찾는 시간을 줄이고,
초보자도 쉽게 업무 정보를 확인할 수 있도록
**가볍고 직관적인 학교 행정실용 키박스**를 목표로 제작했습니다.
