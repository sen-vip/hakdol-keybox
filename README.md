# 우리학교 키박스🔑

### Hakdol Keybox

학교 행정실에서 자주 사용하는 사이트 계정, 비밀번호, 학교 공통 정보, 은행 계좌 정보, 결제 수단 정보를
**빠르게 찾고 복사할 수 있는 로컬 웹앱**입니다.

👉 **바로 사용하기**
https://sen-vip.github.io/hakdol-keybox/

---

## 📌 소개

**우리학교 키박스🔑**는 학교 행정실에서 반복적으로 확인하는 각종 계정 정보를 한곳에 정리해두고,
검색과 복사 버튼으로 빠르게 사용할 수 있도록 만든 업무 보조 도구입니다.

사이트 계정, ID, PW, 은행 계좌, 카드 정보, 학교 공통 정보를 매번 문서나 메모장에서 찾지 않고
브라우저 안에서 바로 확인하고 복사할 수 있습니다.

---

## ✨ 주요 기능

### 1. 빠른복사

자주 쓰는 항목을 즐겨찾기로 등록하면 상단의 **빠른복사 영역**에서 바로 확인할 수 있습니다.

* 즐겨찾기 항목만 자동 표시
* ID / PW 복사 버튼 제공
* 별표 버튼으로 즐겨찾기 등록 및 해제
* 빠른복사 항목 개수 표시

---

### 2. 사이트 계정 관리

학교 업무에 자주 사용하는 사이트 계정을 정리할 수 있습니다.

* 사이트명
* ID
* PW
* URL
* 메모
* 즐겨찾기 여부

각 항목에는 **ID / PW 복사 버튼**과 **사이트 열기 버튼**이 있어
계정 정보를 빠르게 복사하거나 해당 사이트로 이동할 수 있습니다.

---

### 3. 학교 공통 정보 관리

학교에서 자주 쓰는 공통 정보를 따로 정리할 수 있습니다.

예시:

* 사업자등록번호
* 고유번호
* 대표 전화번호
* 팩스번호
* 주소
* 기관명
* 기타 자주 쓰는 행정 정보

---

### 4. 은행 계좌 정보 관리

학교 계좌 정보를 보기 쉽게 정리할 수 있습니다.

* 은행명
* 계좌번호
* 예금주
* 메모

계좌번호는 복사 버튼으로 바로 복사할 수 있습니다.

---

### 5. 결제 수단 관리

카드나 결제 관련 정보를 정리할 수 있습니다.

* 카드명
* 카드번호 일부
* 비고
* 사용처 메모

업무 중 자주 확인하는 결제 수단 정보를 따로 모아둘 수 있습니다.

---

### 6. 검색 기능

상단 검색창에서 원하는 정보를 바로 찾을 수 있습니다.

검색 대상에는 사이트명, ID, URL, 메모, 학교 정보, 계좌 정보 등이 포함됩니다.

---

### 7. 엑셀 다운로드 / 업로드

데이터를 엑셀 형식으로 내려받거나 다시 업로드할 수 있습니다.

* **엑셀 다운**: 현재 입력된 정보를 엑셀 파일로 저장
* **엑셀 업로드**: 엑셀 파일을 불러와 데이터 반영

일반 사용자에게 어려운 JSON 용어 대신
실제 업무에서 익숙한 **엑셀 다운 / 엑셀 업로드** 표현을 사용했습니다.

---

### 8. JSON 백업 / 불러오기

전체 데이터를 백업하거나 복원할 수 있습니다.

* JSON 백업
* JSON 불러오기

엑셀 관리와 별도로, 전체 데이터를 안전하게 보관하고 옮길 때 사용할 수 있습니다.

---

## 🔒 저장 방식

우리학교 키박스는 서버에 데이터를 저장하지 않습니다.

모든 정보는 사용자의 **현재 PC 브라우저 localStorage**에만 저장됩니다.

즉,

* 외부 서버로 데이터 전송 없음
* 별도 로그인 없음
* 데이터베이스 사용 없음
* GitHub Pages에서 바로 실행 가능
* 같은 PC, 같은 브라우저에서 다시 열면 데이터 유지

다만 브라우저 캐시 삭제, localStorage 삭제, 다른 PC 사용 시 데이터가 보이지 않을 수 있으므로
중요한 정보는 반드시 **엑셀 다운** 또는 **JSON 백업**으로 따로 보관하는 것을 권장합니다.

## 🖥️ 사용 방법

### 1. 웹앱 접속

아래 링크로 접속합니다.

https://sen-vip.github.io/hakdol-keybox/

---

### 2. 필요한 정보 입력

각 영역에서 필요한 정보를 입력합니다.

* 사이트 계정
* 학교 공통 정보
* 은행 계좌 정보
* 결제 수단

입력한 정보는 현재 브라우저에 자동으로 저장됩니다.

---

### 3. 자주 쓰는 항목 즐겨찾기

사이트 계정 카드의 별표를 누르면
해당 계정이 상단 **빠른복사** 영역에 표시됩니다.

다시 별표를 누르면 즐겨찾기가 해제됩니다.

---

### 4. 복사 버튼 사용

ID, PW, 계좌번호 등 자주 복사하는 정보는
각 항목의 복사 버튼을 눌러 바로 복사할 수 있습니다.

---

### 5. 백업하기

업무용 PC 변경, 브라우저 초기화, 데이터 정리 전에는
반드시 백업 파일을 내려받아 보관하세요.

추천 백업 방식:

1. 엑셀 다운
2. JSON 백업

## 만든 이유

학교 행정실에서는 반복적으로 찾아야 하는 정보가 많습니다.

어떤 정보는 엑셀에 있고,
어떤 정보는 메모장에 있고,
어떤 정보는 누군가의 책상 위 파일에 있고,
또 어떤 정보는 기억에만 의존하기도 합니다.

우리학교 키박스는 그런 정보를 한곳에 정리해서
학교 행정실의 작은 반복 업무를 조금 더 가볍게 만들기 위해 제작했습니다.

**찾는 시간은 줄이고, 복사는 빠르게.**
**우리학교 정보는 우리 PC 안에서 안전하게.**
