# 우리학교 키박스🔑

학교 행정실에서 자주 사용하는 사이트 계정, 비밀번호, 은행계좌 정보 등을 빠르고 안전하게 관리하기 위한 로컬 웹앱입니다.

별도의 서버나 데이터베이스 전송 없이 브라우저(LocalStorage)에만 저장되므로 설치가 간편하며, GitHub Pages 환경에서도 바로 사용할 수 있습니다.

---

## 주요 기능

### 🔍 빠른 검색

* 사이트명, URL, 아이디, 메모 검색
* 입력 즉시 실시간 필터링

### 🔑 계정 관리

* 사이트 계정 등록·수정·삭제
* 아이디·비밀번호 관리
* 비밀번호 마스킹 처리

### 📋 원클릭 복사

* 아이디 복사
* 비밀번호 복사
* 계좌번호 복사
* URL 바로가기

### ⭐ 즐겨찾기

* 자주 사용하는 계정 즐겨찾기 등록
* 상단 빠른복사 영역 제공

### 🏫 학교 공통 정보 관리

* 학교 기본 정보 저장
* 기관 정보 및 메모 관리

### 🏦 은행 계좌 관리

* 학교 계좌 정보 등록
* 계좌번호 즉시 복사

### 💳 결제 수단 관리

* 카드 정보 및 결제 관련 메모 관리

### 💾 백업 및 복원

* JSON 파일 내보내기
* JSON 파일 불러오기
* 데이터 백업 지원

### 🖨 출력 지원

* A4 인쇄 최적화
* 행정실 비치용 출력 가능

## 보안 정책

우리학교 키박스는 개인정보 보호와 보안을 최우선으로 설계되었습니다.

* 서버 저장 없음
* 회원가입 없음
* 로그인 없음
* 외부 데이터 전송 없음
* 모든 데이터는 사용자의 브라우저 LocalStorage에만 저장

※ 브라우저 데이터 삭제 시 저장된 정보도 함께 삭제될 수 있으므로 정기적인 백업을 권장합니다.
