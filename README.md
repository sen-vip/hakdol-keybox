# 🔑 School Keybox

학교 행정실에서 자주 사용하는 사이트 계정, 비밀번호, 학교 공통 정보, 은행 계좌, 결제 수단을 한곳에 정리하고 빠르게 복사할 수 있는 로컬 웹앱입니다.

> 서버 전송 없음 · 로그인 없음 · 외부 DB 저장 없음  
> 입력한 데이터는 사용자의 PC 브라우저에만 저장됩니다.

## 🔗 바로가기

- 배포 사이트: https://sen-vip.github.io/school-keybox/
- GitHub 저장소: https://github.com/sen-vip/school-keybox

기존 `hakdol-keybox` 저장소를 이름 변경해서 사용할 경우, GitHub Pages 주소도 저장소 이름에 맞춰 `school-keybox`로 바뀝니다.

## ✨ 주요 기능

### 빠른 검색

사이트명, 아이디, 메모, 계좌명, 학교 정보 등을 검색해 필요한 정보를 빠르게 찾을 수 있습니다.

- 검색어 입력 시 결과 개수 표시
- 검색창 X 버튼으로 검색어 빠른 삭제
- 검색 결과가 없을 때 안내 문구 표시
- 빠른복사 영역도 검색어 기준으로 함께 필터링

### 빠른복사

자주 쓰는 계정을 즐겨찾기에 추가하면 상단 빠른복사 영역에서 바로 사용할 수 있습니다.

- 별표 버튼으로 빠른복사 추가 및 해제
- ID 버튼 클릭 시 아이디 복사
- PW 버튼 클릭 시 비밀번호 복사
- 검색 중에는 검색어와 일치하는 빠른복사 항목만 표시

### 사이트 계정 관리

학교 업무에 필요한 사이트 계정 정보를 직접 추가, 수정, 삭제할 수 있습니다.

- 사이트명, ID, PW, 메모, URL 관리
- ID/PW 원클릭 복사
- URL이 있는 경우 사이트 바로 열기
- 행 추가 및 행 삭제
- 드래그로 순서 이동
- 비밀번호 표시 방식 선택

### 학교 정보 관리

학교에서 반복적으로 사용하는 기본 정보를 정리할 수 있습니다.

예시 항목:

- 학교 메일
- 사업자등록번호
- 학교 기관코드
- 나이스 조직코드
- 인증서 관련 정보
- 공용 폴더 정보
- 기타 행정실 공통 정보

### 은행 계좌 정보 관리

학교에서 자주 사용하는 계좌 정보를 따로 정리할 수 있습니다.

- 계좌명
- 계좌번호
- 계좌 PW
- 계좌 관련 메모

### 결제 수단 관리

카드 및 결제 관련 정보를 정리할 수 있습니다.

- 카드명
- 카드번호
- 결제 PW
- 인증서 관련 메모
- 필요 시 복사 기능 활용

## 💾 저장 방식

School Keybox는 서버에 데이터를 저장하지 않습니다.

데이터는 사용자의 브라우저 `localStorage`에만 저장됩니다.

- 같은 PC와 같은 브라우저에서만 저장 내용이 유지됩니다.
- 다른 PC에서는 자동으로 보이지 않습니다.
- 브라우저 데이터를 삭제하면 저장 정보가 사라질 수 있습니다.
- 중요한 정보는 엑셀 또는 JSON으로 별도 백업하는 것을 권장합니다.

## 🛠️ 관리 도구

| 기능 | 설명 |
|---|---|
| 이 PC에 저장 | 현재 입력한 내용을 이 브라우저에 저장합니다. |
| PC 저장내용 불러오기 | 이전에 저장한 내용을 다시 불러옵니다. |
| 인쇄/PDF | 현재 내용을 인쇄하거나 PDF로 저장합니다. |
| 엑셀 다운 | 현재 데이터를 엑셀 파일로 내려받습니다. |
| 엑셀 업로드 | 엑셀 파일을 업로드해 데이터를 불러옵니다. |
| 기본양식 다운로드 | 입력용 기본 엑셀 양식을 다운로드합니다. |
| JSON 백업 | 전체 데이터를 JSON 파일로 백업합니다. |
| JSON 불러오기 | 백업한 JSON 파일을 다시 불러옵니다. |
| 저장정보 초기화 | 이 PC 브라우저에 저장된 정보를 초기화합니다. |

## 🚀 사용 방법

1. 사이트에 접속합니다.
2. 사이트 계정, 학교 정보, 은행 계좌, 결제 수단을 입력합니다.
3. `이 PC에 저장` 버튼을 눌러 현재 브라우저에 저장합니다.
4. 자주 쓰는 계정은 별표를 눌러 빠른복사에 추가합니다.
5. 검색창에서 필요한 정보를 검색합니다.
6. ID 또는 PW 복사 버튼을 눌러 필요한 값을 복사합니다.
7. 필요할 때 엑셀 또는 JSON으로 백업합니다.

## 📊 엑셀 업로드 형식

엑셀 업로드는 `기본양식 다운로드` 파일을 기준으로 사용하는 것을 권장합니다.

기본 시트 구성:

- `기본정보`
- `계정입력`

계정입력 시트 권장 열:

| 사이트명 | 아이디 | 비밀번호 | 메모 | URL | 즐겨찾기 |
|---|---|---|---|---|---|
| 예시 사이트 | example_id | example_pw | 업무용 메모 | https://example.com | Y |

즐겨찾기 열에 `Y`를 입력하면 빠른복사 항목으로 등록됩니다.

## 🆕 업데이트 내용

최근 버전에서는 사용자가 헷갈릴 수 있는 저장, 글씨, 화면 표시 문제를 정리했습니다.

- `이 PC에 저장` 후 새로고침 시 저장 내용 자동 복원
- 저장정보 초기화 시 브라우저 저장값까지 함께 삭제
- ID/PW 및 정보 입력값 글씨 가독성 개선
- 학교 정보, 은행 계좌, 결제 수단, 쇼핑몰 카드 입력 영역의 기본 textarea 손잡이 제거
- 탭 아이콘에 열쇠 파비콘 적용
- 검색창 X 버튼 및 검색 지우기 흐름 개선
- 빠른복사 영역 검색 필터링 개선

## 🔒 보안 안내

이 앱은 학교 업무 편의를 위한 로컬 저장 도구입니다.

- 공용 PC에서는 사용을 권장하지 않습니다.
- 사용 후 브라우저 저장 데이터를 관리해 주세요.
- 중요한 계정 정보는 학교 내부 보안 기준에 따라 관리해 주세요.
- 배포 전 예시 데이터에 실제 아이디, 비밀번호, 인증서 정보가 들어가지 않았는지 반드시 확인해 주세요.
